Gestion de Torneos

¡Bienvenido {{user.username}} a mi proyecto de administración de torneos!

Este es un proyecto personal en django, y representa una parte importante de mi portfolio como desarrollador.

Caracteristicas principales de la aplicacion

  • Registro y Login de Usuarios: Sistema de registro y autenticación que permite a los usuarios crear cuentas y acceder de manera segura a sus torneos.

  • Gestion de clubes: Funcionalidades completas de CRUD (Crear, Leer, Actualizar, Eliminar) para facilitar la gestión de los clubes de la organizacion.

  • Gestion de padron: Funcionalidades completas de CRUD (Crear, Leer, Actualizar, Eliminar) para la gestión de los jugadores de cada club.

  • Gestion de torneos: Funcionalidades para crear torneos, generando un fixture del mismo con todos los clubes asociado al usuario, permitiendo actualizar el resultado de todos los partidos del fixture.

  • Tablas de posiciones: Funcionalidades para calcular las tablas de posiciones segun el resultado de los partidos, facilitando el seguimiento de los torneos.

  • Base de Datos SQLite3: Utilización de SQLite3 como base de datos para almacenar de manera segura y eficiente la información. En produccion se utiliza POSTGRESQL.

  • bootstrap: Utilizacion de bootstrap para los estilos de la aplicacion.

Gracias por su visita y su interés.

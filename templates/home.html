Características principales de la aplicación

Registro y Login de Usuarios: Sistema de registro y autenticación que permite a los usuarios crear cuentas y acceder de manera segura a sus torneos.

Gestión de clubes: Funcionalidades completas de CRUD (Crear, Leer, Actualizar, Eliminar) para facilitar la gestión de los clubes de la organización.

Gestión de padrón: Funcionalidades completas de CRUD (Crear, Leer, Actualizar, Eliminar) para la gestión de los jugadores de cada club.

Gestión de torneos: Funcionalidades para crear torneos, generando un fixture del mismo con todos los clubes asociados al usuario, permitiendo actualizar el resultado de todos los partidos del fixture.

Tablas de posiciones: Funcionalidades para calcular las tablas de posiciones según el resultado de los partidos, facilitando el seguimiento de los torneos.

SQLite3 y Bootstrap: Utilización de SQLite3 como base de datos y Bootstrap para los estilos de la aplicación. En producción utiliza PostgreSQL como base de datos.
